# ai-pdf-filler

Codex skill repository for AI PDF form filling with `simplicity-cli`.

## Included files

- `SKILL.md` - skill behavior and workflow rules
- `references/commands.md` - command cheat sheet
- `agents/openai.yaml` - UI metadata

## Quick usage

1. Install the CLI: `uv tool install ai-pdf-filler` (or `pip install ai-pdf-filler`)
2. Save API key once: `simplicity-cli login`
3. Invoke the skill as `$ai-pdf-filler-cli` in Codex prompts.
